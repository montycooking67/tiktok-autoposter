Privacy Policy

Last updated: 2025-12-11

1. Scope

This Privacy Policy describes how the "TikTok Auto Uploader" desktop tool ("the Tool") handles data. The Tool is used only by me, the developer, on my own computer.

2. Personal Data

The Tool does not collect personal data from third-party users. It only processes:

  • Media files (images and videos) stored locally on my own device
  • Captions and metadata that I manually define
  • Authentication tokens for my own TikTok account, stored locally in configuration files or environment variables

3. Data Storage

All configuration files and media files are stored locally on my own computer or backup devices under my control. No data is stored on servers that belong to this Tool.

4. Data Sharing

The Tool sends media files and captions to TikTok via the official API for the purpose of publishing posts on my TikTok account. Apart from TikTok, no other third parties receive any data.

5. Security

API keys and access tokens are stored in local configuration files or environment variables. I take reasonable steps to keep my device secure (for example using passwords and up-to-date software).

6. Changes

I may update this Privacy Policy from time to time and will update the "Last updated" date at the top of this page.
